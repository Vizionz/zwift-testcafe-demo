# zwift-testcafe-demo

Simple Login for Zwift.com

## Requirements

- Java 12.0.2+ https://www.oracle.com/java/technologies/javase-downloads.html
- NodeJS 10.16.3+ https://nodejs.org/en/

## To Run

1. Install Node Modules - Execute `npm install` from repo home in terminal/commandline.
2. Run Tests - `$(npm bin)/testcafe chrome ./tests
